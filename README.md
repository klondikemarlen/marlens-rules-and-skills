# OpenCode Global Rules

Version-controlled global OpenCode instructions for `~/.config/opencode/AGENTS.md`.

## Use

Link the tracked rules into OpenCode:

```bash
ln -sf /home/marlen/code/klondikemarlen/opencode-global-rules/AGENTS.md /home/marlen/.config/opencode/AGENTS.md
```

Restart OpenCode after changing this file. OpenCode loads instructions at startup.

## Files

- `AGENTS.md` - global agent instructions loaded by OpenCode
